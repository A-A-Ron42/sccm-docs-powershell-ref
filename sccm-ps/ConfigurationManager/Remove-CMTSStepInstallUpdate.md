# Remove-CMTSStepInstallUpdate

## SYNOPSIS
Removes a TS step install update.

## SYNTAX

### ByValue (Default)
```
Remove-CMTSStepInstallUpdate [-InputObject] <IResultObject> [-StepName <String>] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ById
```
Remove-CMTSStepInstallUpdate [-TaskSequenceId] <String> [-StepName <String>] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByName
```
Remove-CMTSStepInstallUpdate [-TaskSequenceName] <String> [-StepName <String>] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
